# Contributing

Thanks for your interest in improving Contrix.

## Ways to Contribute
- bug reports
- feature proposals
- documentation improvements
- code contributions

## Basic Flow
1. Open an issue for major feature changes
2. Fork and create a branch
3. Keep changes focused and tested
4. Submit a clear pull request

## Suggested Areas
- provider adapter expansion
- richer schema UX
- runtime policy improvements
- docs and examples
